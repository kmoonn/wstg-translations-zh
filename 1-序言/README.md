## 欢迎

> 由于我们专注于渐进式改进，本版本引入了大量更新。我们规范了场景格式，以创造更好的阅读体验，为每个测试场景添加了目标，合并了部分内容，并就一些现代测试主题添加了新的场景。
>
> — Rick Mitchell

OWASP 感谢众多作者、审阅者和编辑的辛勤工作，才有了今天的这份指南。如果您对《测试指南》有任何意见或建议，请随时向我们的[GitHub 仓库](https://github.com/OWASP/wstg/)提交你的问题。
